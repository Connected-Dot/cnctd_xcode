# cnctd_xcode

Xcode/iOS build toolchain wrapper for Rust.

## Installation

```toml
[dependencies]
cnctd_xcode = { git = "https://github.com/Connected-Dot/cnctd_crates.git" }
```

## Features

- Build iOS/macOS projects
- Manage simulators
- Archive and export
- Code signing operations
